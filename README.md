# ALPHA

An AI-powered image gallery that uses Groq cloud models to generate, store, and showcase AI-created images.

## Pages

- **Showcase** (`/`) — Orbiting image gallery with rotating rings
- **Create** (`/create`) — AI chatbot interface for image generation
- **Gallery** (`/gallery`) — Browse all generated images

## Tech Stack

- React 19 + TypeScript 6
- Framer Motion 12 + GSAP 3
- Three.js + React Three Fiber
- Tailwind CSS v4
- Express.js + SQLite

## Commands

```bash
npm run dev        # Start Vite dev server
npm run server     # Start Express API server
npm run dev:all    # Run both concurrently
npm run build      # Production build
```
